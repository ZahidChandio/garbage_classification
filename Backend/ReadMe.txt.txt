Open Project in IDE (VS Code)
Open VS code terminal and run below commands
> python -m venv my_venv
> my_venv\Scripts\active
> python -m pip install numpy
> python -m pip install tensorflow
> python -m pip install keras	
> python -m pip install django
> python -m pip install django-cors-headers
> python -m pip install djangorestframework
> python -m pip install Pillow
> python -m pip install --upgrade keras

Requirements:
Install Python
Install django ( open command line and run this command python -m pip install Django)
Install pip (open command line and run this command curl https://bootst/rap.pypa.io/get-pip.py -o get-pip.py [this is for windows])
